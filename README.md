# Astro Bot

Astro Bot is a lightweight astrology analysis tool built with Python, Gradio, and Swiss Ephemeris.  
It calculates planetary positions, ascendant, houses, moon sign, Manglik status, and Vimshottari Dasha.  
It also generates a clean PDF horoscope report.
